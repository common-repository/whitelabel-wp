== Description ==

Whitelabel for Wordpress is a simple plugin which changes the login color scheme and hide backend wordpress footprints.

**Features**
* Disabled dashboard widget "Wordpress Events and News",
* Hidden help toggle button on top right corner (wp-admin).
* Hidden wordpress logo and other sub-menus of it on top left corner (wp-admin).
* Hidden "Thank you for creating with WordPress." at the bottom (wp-admin).
* Hidden wordpress version at bottom right corner (wp-admin).
* Switched wp-admin backend theme to "Dark Midnight" by default.
* Disabled user color scheme picker.
* Login page designed as "Dark Midnight" theme (wp-login).
* Wordpress logo removed (wp-login).

== Installation ==

Automation Installation
1. Plugin->Add New->Search for "whitelabel for wordpress"
2. Download, install, and activate the plugin.

Manual Installation
1. Download Whitelabel for Wordpress.
2. Upload the plugin directory to your ‘/wp-content/plugins/’ directory, using your favorite method (ftp, sftp, scp, etc…).
3. Activate this plugin from your plugins page.

== Frequently Asked Questions ==

= Problem after activating? =

Deactivate through plugin page or ftp/file manager by simply deleting folder named "whitelabel-wp" inside (/wp-content/plugins) & Report on support.
